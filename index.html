Kết nối Google Apps Script

Dán URL Web App của Google Apps Script vào đây để kết nối dữ liệu.

Hồ sơ

Hướng dẫn nhanh

  1. Mở Google Sheets → Tạo spreadsheet mới
  2. Sao chép Spreadsheet ID từ URL
  3. Mở Extensions → Apps Script
  4. Dán nội dung file Code.gs vào editor
  5. Thay YOUR_SPREADSHEET_ID_HERE bằng ID của bạn
  6. Chạy hàm setupSheets() một lần để tạo cấu trúc
  7. Click Deploy → New deployment → Web App
  8. Chọn "Execute as: Me" và "Who has access: Anyone"
  9. Sao chép URL và dán vào ô API URL ở trên
